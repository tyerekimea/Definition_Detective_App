# 🚀 PlayStore Publication Quick Start\n\n**Everything is ready! Here's what to do next:**\n\n## Step 1: Review Your Assets ✅ (Already Done)\n\n```bash\ncd playstore-assets\nls -la icons/          # 7 app icon sizes\nls -la graphics/       # Feature and promo graphics\ncat app-listing.json   # Review your app metadata\n```\n\n## Step 2: Capture Screenshots (20-30 minutes)\n\n### Option A: Using Emulator\n```bash\n# In another terminal, start the app on emulator:\nnpm run dev:mobile\n\n# Capture screenshot:\nadb shell screencap -p /sdcard/screenshot.png\nadb pull /sdcard/screenshot.png playstore-assets/screenshots/phone/screenshot-1.png\n\n# Repeat for 4-5 different screens\n```\n\n### Option B: Using Physical Device\n1. Connect device via USB\n2. Enable developer mode & USB debugging\n3. Run the app\n4. Press Power + Volume Down to capture\n5. Transfer files to `playstore-assets/screenshots/phone/`\n\n### Required Screenshots\n- Minimum: 2 screenshots\n- Recommended: 4-5 screenshots\n- Dimensions: 1080 x 1920 pixels (9:16 aspect ratio)\n- Format: PNG or JPEG\n- File names: `screenshot-1.png`, `screenshot-2.png`, etc.\n\n## Step 3: Optional - Add Captions to Screenshots\n\n```bash\ncd playstore-assets\npython3 add_captions_to_screenshots.py\n```\n\nThis will create captioned versions in `screenshots/phone-captioned/`\n\n## Step 4: Build Production APK (15-20 minutes)\n\n```bash\n# From project root:\nnpm run build:mobile:prod\n\n# Output: app-release.apk (ready for upload)\n```\n\n## Step 5: Create PlayStore Listing (30-45 minutes)\n\n1. **Go to Google Play Console**\n   - https://play.google.com/console\n   - Sign in with your Google account\n   - Create new app or select existing\n\n2. **Fill Store Listing**\n   - App title: \"Definition Detective\"\n   - Short description: Copy from `app-listing.json`\n   - Full description: Copy from `app-listing.json`\n   - Category: Games → Casual\n   - Content rating: Everyone\n\n3. **Upload Graphics**\n   - App icon: `icons/ic_launcher-playstore-512x512.png`\n   - Feature graphic: `graphics/feature-graphic-1024x500.png`\n   - Screenshots (4-5): `screenshots/phone/screenshot-*.png`\n\n4. **Configure Details**\n   - Contact email: support@traylapps.com\n   - Privacy policy: https://traylapps.com/privacy\n   - Website: https://traylapps.com\n\n5. **Set Pricing**\n   - Select: Free\n   - Supported countries: All or as desired\n\n## Step 6: Upload & Test APK (5-10 minutes)\n\n1. **Upload Release APK**\n   - Go to \"Content\" → \"App bundle and APK\"\n   - Click \"Create new release\"\n   - Upload: `app-release.apk`\n   - Add release notes\n\n2. **Content Rating Questionnaire**\n   - Complete the rating questionnaire\n   - Should take ~5 minutes\n\n## Step 7: Submit for Review (1 minute + 2-5 days waiting)\n\n1. Review all information\n2. Click \"Submit for Review\"\n3. Confirm\n4. **Wait for approval (2-5 business days)**\n\n---\n\n## File Locations Reference\n\n| Item | Location | PlayStore Use |\n|------|----------|---------------|\n| Main Icon | `icons/ic_launcher-playstore-512x512.png` | App icon in listing |\n| Feature Graphic | `graphics/feature-graphic-1024x500.png` | Store listing banner |\n| Screenshots |
