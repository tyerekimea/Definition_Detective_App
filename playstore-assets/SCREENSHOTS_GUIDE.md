# PlayStore Screenshots & Metadata\n\n## Screenshot Guidelines\n\n### Recommended Screenshots (5 screenshots)\n\n1. **Main Game Screen**\n   - Show the core gameplay mechanic\n   - Caption: \"Solve word mysteries\"\n   - Dimensions: 1080 x 1920\n\n2. **Features/Benefits**\n   - Showcase smart hints or learning mode\n   - Caption: \"Smart hints to guide you\"\n   - Dimensions: 1080 x 1920\n\n3. **Social/Leaderboards**\n   - Show competitive elements\n   - Caption: \"Compete on global leaderboards\"\n   - Dimensions: 1080 x 1920\n\n4. **Customization**\n   - Show themes or personalization\n   - Caption: \"Customize your experience\"\n   - Dimensions: 1080 x 1920\n\n5. **Achievements/Progress**\n   - Show progression system\n   - Caption: \"Unlock achievements and rewards\"\n   - Dimensions: 1080 x 1920\n\n## Capturing Screenshots\n\n### Using Android Emulator\n\n```bash\n# Take screenshot via ADB\nadb shell screencap -p /sdcard/screenshot.png\n\n# Pull to computer\nadb pull /sdcard/screenshot.png ./playstore-assets/screenshots/phone/\n\n# Clean up device\nadb shell rm /sdcard/screenshot.png\n```\n\n### Using Physical Device\n\n1. Connect device via USB\n2. Enable developer mode\n3. Enable USB debugging\n4. Run the app\n5. Use device's screenshot button (Power + Volume Down)\n6. Transfer to computer\n7. Resize to 1080 x 1920 if needed\n\n### Using Chrome DevTools (Web Version)\n\n1. Open app in Chrome browser\n2. Toggle device toolbar (F12)\n3. Set device to phone dimensions\n4. Arrange UI as desired\n5. Take screenshot\n6. Crop/resize to 1080 x 1920\n\n## Adding Text Captions to Screenshots\n\n### Using GIMP (Free)\n\n```bash\n# Install GIMP\nsudo apt install gimp\n\n# Open screenshot\ngimp screenshot.png\n\n# Add text:\n# 1. Select Text tool (T)\n# 2. Click on image\n# 3. Type caption\n# 4. Adjust font size (30-50px)\n# 5. Position at top or bottom\n# 6. Export as PNG\n```\n\n### Using Python/PIL (Programmatic)\n\nSee the `add_captions_to_screenshots.py` script included in this directory.\n\n## Screenshot Checklist\n\nBefore uploading to PlayStore:\n\n- [ ] **Dimensions**: All screenshots are 1080 x 1920 pixels\n- [ ] **Format**: PNG or JPEG, no transparency needed\n- [ ] **Quality**: High quality, readable text, clear visuals\n- [ ] **Content**: Shows actual app features (no mockups)\n- [ ] **Quantity**: At least 2 screenshots (recommend 4-5)\n- [ ] **Captions**: Text is clear and readable (max 30 chars)\n- [ ] **Branding**: Consistent with app branding\n- [ ] **Legibility**: Text visible at all screen sizes\n- [ ] **No overlays**: No watermarks or \"tap to start\" messages\n- [ ] **Ordered correctly**: Most important first\n\n## Screenshot Descriptions\n\n### Screenshot 1: \"Solve word mysteries\"\n- Show the main game interface\n- Display the clue and definition format\n- Highlight the fun, engaging design\n\n### Text Overlay Best Practices\n\n1. **Font**: Use bold, sans-serif fonts (e.g., Arial, Roboto)\n2. **Size**: 40-60px for headlines, readable at all sizes\n3. **Color**: Use contrasting colors (white text on dark background, or dark on light)\n4. **Position**: Top or bottom, avoid blocking important content\n5. **Shadow**: Add drop shadow for better readability\n6. **Alignment**: Center or left-aligned, consistent across screenshots\n\n## Example Text Overlays\n\n```\nScreenshot 1:\n  \"Solve word mysteries\"\n  \"What's the definition?\"\n\nScreenshot 2:\n  \"Smart hints\"\n  \"Get Open [Google Play Console](https://play.google.com/console)\n2. Select your app\n3. Go to **Store listing**\n4. Scroll to **Screenshots** section\n5. Click **Add screenshots**\n6. Select phone category (1080x1920)\n7. Upload your 2-8 screenshots\n8. Add captions (optional but recommended)\n9. Save and review\n10. Submit for review\n\n## Common Issues & Solutions\n\n| Issue | Solution |\n|-------|----------|\n| Screenshot too small | Resize to exactly 1080x1920 |\n| Text not readable | Increase font size, add shadow, use contrasting color |\n| Wrong aspect ratio | Crop to 9:16 (1080:1920) |\n| File too large | Compress PNG/JPEG (target <5MB per image) |\n| Unsupported format | Convert to PNG or JPEG |\n\n## File Organization\n\n```\nscreenshots/\n├── phone/\n│   ├── screenshot-1.png           # Solve mysteries\n│   ├── screenshot-2.png           # Smart hints\n│   ├── screenshot-3.png           # Leaderboards\n│   ├── screenshot-4.png           # Themes\n│   └── screenshot-5.png           # Achievements\n├── tablet/\n│   ├── screenshot-1.png\n│   ├── screenshot-2.png\n│   └── screenshot-3.png\n└── watches/\n    └── screenshot-1.png           # (if applicable)\n```\n\n## Next Steps\n\n1.
